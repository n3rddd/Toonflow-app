# Production Agent

短剧漫剧制作专用技能，提供剧本创作、分镜设计、角色设定等全流程指导。

## 何时使用

当用户需要以下帮助时激活此技能：

- 开始制作视频

## 工作指引

1. 理解用户的创作意图，根据项目类型（短剧/漫剧）调整输出风格
2. 遵循标准的剧本格式，包含场景描述、角色动作、对白等要素
3. 保持角色一致性，关注剧情连贯性
4. 输出使用中文
